Introduction
How to Use Ceiling Mounted Projector
Using a projector can allow you to easily show other people how to use a program by allowing them to see everything that you see. The projector can also be used to display a visual aid for a presentation aiding in the effectiveness of your speech. The projector has many uses. All of the connections to the projector, such as the power cable, display cables, and other connection cables will already be made, so no need to worry about them.
Using PowerPoint with Computers
PowerPoint is a simple program that adds a lot of value to presentations and speeches. Using this program successfully can be the difference between a powerful and exciting speech, or a lackluster and boring speech.
Where to go Incase of an Emergency
In the event that there is an emergency in the building and you are required to leave the building you will need to find the nearest exit and get at least 30 feet away from the building.
